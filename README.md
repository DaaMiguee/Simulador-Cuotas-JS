# Simulador de cuotas
Un simulador que sirve para calcular las cuotas dependiendo del monto y las cuotas ingresadas.
Se actualizó para mejorar la funcionalidad y apariencia de la web.

### Tecnologias usadas:
 - HTML
 - CSS
 - JavaScript
